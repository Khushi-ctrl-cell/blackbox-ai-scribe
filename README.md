# ✍️ BlackBox AI Scribe

An AI-powered writing assistant that generates, edits, and enhances written content using modern AI technologies.

## 🚀 Features

* AI text generation
* Content summarization
* Grammar correction
* Smart rewriting

## 🛠 Tech Stack

JavaScript
React
AI APIs

## ⚙ Installation

```
git clone https://github.com/Khushi-ctrl-cell/blackbox-ai-scribe
cd blackbox-ai-scribe
npm install
npm start
```

## 🎯 Goal

To assist writers, students, and professionals in generating high-quality content efficiently.
